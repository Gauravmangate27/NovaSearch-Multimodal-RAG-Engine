Hybrid (FAISS + Elasticsearch) · OpenAI · FastAPI

Overview

Semantic search across text and images using OpenAI embeddings, CLIP/GPT-4 Vision, hybrid retrieval (FAISS + Elasticsearch), and LangChain orchestration served via FastAPI.

Open API Docs View README

Quick Start

1) Set OPENAI_API_KEY in .env
2) Install deps: pip install -r requirements.txt
3) Run API: uvicorn src.api.main:app --reload

Capabilities

Multimodal (Text + Image) OpenAI Embeddings GPT-4 Vision Descriptions FAISS Dense Search Elasticsearch Sparse Hybrid Ranking LangChain LLM Flows FastAPI REST

Endpoints

GET /health — status & indexed count

POST /index — add documents

POST /search — hybrid retrieval

POST /ingest-directory — bulk ingest

Data Flow

Ingest → Embed (OpenAI/CLIP) → Index (FAISS + ES) → Hybrid Search → LangChain answer/summarize → FastAPI response.

Run Locally

Activate venv, install requirements, set .env, then:

uvicorn src.api.main:app --reload

Example Search

POST /search with:

{ "query": "What is machine learning?", "k": 5, "hybrid": true }

Tech Stack

FastAPI · LangChain · OpenAI · FAISS · Elasticsearch · Pydantic · pytest
